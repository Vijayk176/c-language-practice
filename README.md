# Fundamentals of Programming (C Language)

This repository contains my practice files and projects developed while learning the C programming language. It serves as a foundation for my understanding of low-level programming and computer science basics.

## 🚀 Key Concepts Covered
- **Basics:** Data types, Operators, and Input/Output (`printf`, `scanf`).
- **Control Flow:** If-Else statements, Switch-Case, and Loops (For, While, Do-While).
- **Functions:** Modular programming and Scope of variables.
- **Arrays & Strings:** 1D and 2D arrays (like my `2darray.c` scripts) and string manipulation.
- **Pointers:** Memory addresses and pointer arithmetic.
- **Structures:** Creating custom data types.
- **File Handling:** Reading from and writing to files.

## 📂 Folder Structure
- `/Basics`: Simple logic and math problems.
- `/Arrays`: 1D and 2D array implementations.
- `/Logic-Building`: Pattern printing and complex loops.
